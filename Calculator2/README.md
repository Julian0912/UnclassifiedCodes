# Calculator2  
#### 作者：Julian  
### 总说明：  
_认真学习了栈后对 计算器 进行了重写_  
### 改进：  
* 栈的实现更标准  
* 优化了代码容量  
* 增加了详细的注释  
### 缺陷：  
* 不支持负数和小数运算  
* 不支持表达式合法检查
* 对含除法的运算有潜在bug
* 仅支持加减乘除和括号运算

### 使用说明：  
```python
>>> c = Calculator('10+2*41')  # 创建计算器对象，并输入算数式作为参数
>>> res = c.calculate()  # 执行计算
>>> print(res)  # 输出结果
92
```
